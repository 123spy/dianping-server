# 去哪评后端

## 项目简介

这是一个本地生活点评与优惠券系统后端，主要面向“找店、评分评论、收藏、领券下单、后台管理”这些核心业务场景。  
项目在原有店铺与用户体系上，又补充了短信验证码登录、Redis GEO 附近店铺查询、启动全量同步与定时增量同步等能力。

## 技术栈

- Java 21
- Spring Boot 3
- Spring MVC
- MyBatis-Plus
- MySQL
- Redis
- RabbitMQ
- Spring AOP
- Knife4j / OpenAPI 3
- Lombok
- Hutool
- 阿里云短信服务 SDK
- 腾讯云 COS

## 主要业务功能

- 用户注册、登录、退出登录、获取当前登录用户、个人资料修改
- 账号密码登录、手机号验证码登录、短信验证码发送与频控
- 店铺分类查询、店铺分页查询、关键词搜索、店铺详情展示
- 店铺评论、店铺评分、店铺收藏、我的收藏
- 优惠券查询、优惠券下单、支付、取消、我的订单、我的优惠券
- 后台用户管理、店铺管理、评论管理、评分管理、分类管理、优惠券管理、订单管理
- 热门评分榜单
- 基于 Redis GEO 的附近店铺查询

## 项目中用到的关键技术点

- 使用 Redis 对店铺详情、分页列表、热门评分榜单做缓存设计
- 使用 Redis ZSet 维护热门店铺评分榜单
- 使用 Redis GEO 存储店铺坐标并实现附近店铺查询
- 启动时执行店铺坐标全量同步，定时执行增量同步
- 使用 Redis + Lua 处理优惠券抢购场景下的库存校验和一人一单控制
- 使用 RabbitMQ 对订单创建做异步削峰
- 接入阿里云短信服务，实现手机号验证码登录
- 对短信发送增加冷却时间和自然日次数限制
